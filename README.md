#### Tetris说明文档

Hi，这是我在Github上一个关于俄罗斯方块的项目，欢迎star和fork，下面做一个简短的介绍。

本项目基于Java开发，SqlServer做数据库，JDBC实现数据库与软件通信，界面主要采用Java的awt,swing包开发。

##### 项目特色:

1.可切换主题。2.可自定义控制按键。3.每升一级会自动切换一张背景图。4.每升一级会加快方块下落速度。
